 - "사장님" (형님 X)
 - "발주를 6개월 먼저"
 - "정부 공인 데이터 정식 활용"
 - "사전 모집 30명 한정"
 - "1개월 무료 + 평생 30% 할인"

**쓰지 말 것 (카피캣 노출 방지 — 가장 중요)**
 - 이모지 (모든 SNS 콘텐츠에서 픽토/아이콘만 사용)
 - 구체적 데이터 출처 (정부 부처명, 포털명, API 명, 데이터셋 이름 등)

 - 구체적 임계치 (충당금 5억, 사이클 15년, 노후 30년 등 X)
 - 정확한 단지 수치 (1.7만 / 17,000 등 X — "수도권 전 단지" 로)
 - 우리 공종 분류 ("6대 공종" / 6개 공종명 나열 X) — "다공종" 으로
 - 수집 주기 ("매일 새벽" / "24시간" 등 구체적 X) — "정기적으로"
 - 영어 약어 (SaaS, B2B, MRR 등 — 사장님들 거부감)
 - "혁신" / "디스럽트" — 진부함
