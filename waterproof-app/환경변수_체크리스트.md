# 환경변수 체크리스트 (Vercel 입력용)

> Vercel 화면 옆에 띄워놓고, 하나씩 복사해서 채워가세요.
> ✅ 표시 = Vercel 에 입력 완료한 것

---

## 🔑 필수 — 6개 (이게 없으면 앱이 안 돕니다)

### 1. NEXT_PUBLIC_SUPABASE_URL
- [ ] 입력 완료
- **얻는 곳**: supabase.com → 우리 프로젝트 → 좌측 ⚙️ Settings → API → "Project URL"
- **형태**: `https://abcdefghij.supabase.co`
- **값**: ___________________________________

### 2. NEXT_PUBLIC_SUPABASE_ANON_KEY
- [ ] 입력 완료
- **얻는 곳**: 같은 페이지 → "Project API keys" → **anon  public**
- **형태**: 매우 긴 문자열 (eyJhbGc... 로 시작)
- **값**: ___________________________________

### 3. SUPABASE_SERVICE_ROLE_KEY 🔒
- [ ] 입력 완료
- **얻는 곳**: 같은 페이지 → "Project API keys" → **service_role** (Reveal 클릭)
- **형태**: 매우 긴 문자열 (eyJhbGc... 로 시작, anon 키랑 비슷하게 생김)
- ⚠️ 이건 노출되면 DB 가 통째로 털림. GitHub 에 절대 커밋 X.
- **값**: ___________________________________

### 4. ANTHROPIC_API_KEY
- [ ] 입력 완료
- **얻는 곳**: console.anthropic.com → 좌측 API Keys → Create Key
- **형태**: `sk-ant-api03-xxxxx...`
- **참고**: 무료 크레딧 $5 자동 제공, 그 후 종량제
- **값**: ___________________________________

### 5. DATA_GO_KR_KEY
- [ ] 입력 완료
- **얻는 곳**: data.go.kr 로그인 → 마이페이지 → 인증키 확인 → 사용중인 키
- **형태**: 알파벳+숫자 50자 정도
- **신청한 API**: "공동주택 단지 정보" + "유지관리 이력" + "충당금" (3종 다 신청해야 함)
- **값**: ___________________________________

### 6. NEXT_PUBLIC_URL
- [ ] 1차 입력 완료 (placeholder)
- [ ] 2차 수정 완료 (실제 URL)
- **1차**: `https://placeholder.com` (배포 전이라 임시로)
- **2차**: 배포 후 Vercel 이 알려주는 URL 로 교체
  - 예: `https://baljuup-myname.vercel.app`
- **값**: ___________________________________

## Vercel 입력 시 환경(Environment) 선택 가이드

각 변수를 Add 할 때 옆에 체크박스 3개가 있습니다:

| 변수 | Production | Preview | Development |
|---|:---:|:---:|:---:|
| 모든 `NEXT_PUBLIC_*` | ✅ | ✅ | ✅ |
| `SUPABASE_SERVICE_ROLE_KEY` | ✅ | ✅ | ❌ |
| `ANTHROPIC_API_KEY` | ✅ | ✅ | ❌ |
| `DATA_GO_KR_KEY` | ✅ | ✅ | ❌ |
| `TOSS_SECRET_KEY` | ✅ | ✅ | ❌ |
| `KAKAO_*` | ✅ | ✅ | ❌ |

> 헷갈리면 그냥 **3개 다 체크**해도 됩니다. 큰 문제는 없어요.

## 마지막 점검 — 다 입력했는지 확인

Vercel → Settings → Environment Variables 화면에서 다음 6개가 모두 보여야 정상:

```
✅ NEXT_PUBLIC_SUPABASE_URL
✅ NEXT_PUBLIC_SUPABASE_ANON_KEY
✅ SUPABASE_SERVICE_ROLE_KEY
✅ ANTHROPIC_API_KEY
✅ DATA_GO_KR_KEY
✅ NEXT_PUBLIC_URL
```

6개 모두 있으면 → Deployments 탭 → ... → **Redeploy** 클릭하면 끝.
