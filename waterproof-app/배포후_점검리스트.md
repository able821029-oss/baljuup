# 배포 후 점검 리스트

> 배포 성공 = URL 발급 받은 그 순간이 아니라, **실제 핵심 기능이 다 돌아가는지 확인했을 때**입니다.
> 아래 6가지를 라이브 URL 에서 직접 눌러보고 모두 ✅ 가 되면 진짜 출시 준비 완료.

---

## 1️⃣ 화면이 디자인대로 보이는가

배포 URL 접속 후 5분 안에 확인:

- [ ] 로그인 페이지 — 좌측 파랑 보더 + "발주Up" 로고 + 입력칸 모두 정상
- [ ] 회원가입 페이지 — 6개 입력칸 (대표자명/사업체명/휴대폰/이메일/비밀번호/지역) 디자인대로 정렬됨
- [ ] 모바일에서 접속해도 깨지지 않음 (브라우저 우클릭 → 검사 → 휴대폰 아이콘으로 미리보기 가능)

❌ 깨져 있다면 → Vercel 빌드 로그 확인 (Deployments → 최신 → View Build Logs)

---

## 2️⃣ 회원가입 + 로그인이 끝까지 되는가

가장 중요. 이게 안 되면 신규 사용자 1명도 못 받음.

- [ ] `/signup` 에서 가짜 이메일로 가입 시도
- [ ] "가입 신청을 받았습니다" 화면 도착
- [ ] 메일함에 Supabase 인증 메일 수신 (스팸함도 확인)
- [ ] 메일 안의 링크 클릭 → 배포된 사이트로 이동 (❗ localhost 로 가면 안 됨)
- [ ] 자동으로 대시보드(`/dashboard`) 진입

❌ 이메일이 안 오면:
- Supabase → Authentication → Email Templates 에 "Confirm signup" 활성화 확인
- 무료 SMTP 한도(1시간 4건)에 걸렸을 수 있음. Resend 무료 연결 추천.

❌ localhost 로 가면:
- Supabase → Authentication → URL Configuration → Site URL 이 진짜 배포 URL 인지 확인

---

## 3️⃣ 대시보드가 데이터를 보여주는가

- [ ] 4개 카드(모니터링 단지 / 즉시 접촉 / 이번 달 알림 / 수주 성공) 가 숫자로 보임
- [ ] 첫 가입 직후라면 모두 0 또는 빈 상태가 정상
- [ ] 좌측 사이드바 5개 메뉴(대시보드/단지/제안서/알림/설정) 클릭 시 페이지 전환됨

❌ 모두 빈 화면이면:
- DB 데이터가 없는 게 정상 (수집을 1번도 안 했으니)
- 다음 단계에서 수집 1회 실행 (아래 4번 참조)

---

## 4️⃣ 공공데이터 수집이 되는가

- [ ] GitHub 저장소 → Actions 탭 클릭
- [ ] **Weekly Collect Complexes** 워크플로우 우측 **Run workflow** → main → Run
- [ ] 5~10분 후 완료 → 초록 ✅ 표시
- [ ] 대시보드 새로고침 → 모니터링 단지 숫자가 6,000건 이상으로 바뀜

❌ 워크플로우 실패 시:
- Actions → 실패한 run 클릭 → 빨간 X 표시된 step 의 로그 확인
- 보통 `DATA_GO_KR_KEY` 또는 `SUPABASE_SERVICE_ROLE_KEY` 가 GitHub Secrets 에 없는 경우

---

## 5️⃣ 단지 상세 + AI 제안서가 되는가

가장 핵심 기능 검증:

- [ ] 단지 목록(`/complexes`) 에서 점수 높은 단지 하나 클릭
- [ ] 단지 상세 페이지에서:
  - AI 예측 점수 카드 표시됨
  - 유지관리 이력 타임라인 표시됨
  - 충당금 차트 표시됨
- [ ] 하단 "제안서 만들기" 버튼 클릭
- [ ] 30초~1분 대기 후 AI 가 작성한 제안서 PDF 다운로드

❌ AI 제안서가 안 만들어지면:
- `ANTHROPIC_API_KEY` 잔액 확인 (console.anthropic.com → Usage)
- Vercel Function 로그 확인 (Deployments → Functions → /api/generate-proposal)

---

## 6️⃣ 휴대폰으로 진짜 사용 가능한가

마지막으로 본인 휴대폰으로 접속해서:

- [ ] 카카오톡 등으로 본인에게 URL 전송 → 클릭해서 열림
- [ ] 회원가입 → 로그인 → 대시보드까지 끊김 없이 진행됨
- [ ] 하단 5탭 네비게이션이 엄지로 누르기 좋게 배치됨
- [ ] 스크롤이 부드럽고, 글자가 모두 읽힘
