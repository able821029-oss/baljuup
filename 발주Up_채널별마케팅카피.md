> 원칙: 합쇼체 고수 / AI·머신러닝 등 기술 용어 금지 / 과장 금지 / 신뢰감 우선

## 1. 네이버 카페 쪽지 마케팅

### 기본 쪽지 (최초 접촉)

```
안녕하십니까.

방수공사 영업 관련 정보 공유드리려고 연락드렸습니다.

저희는 수도권 아파트 단지 중 방수공사가 임박한 곳을
공고 나오기 3~6개월 전에 미리 알려드리는 서비스를 만들었습니다.

이번 달 수도권 베타 사용자를 무료로 모집하고 있습니다.
관심 있으시면 아래 링크 한 번만 눌러봐 주십시오.

[발주Up 베타 신청 링크]

감사합니다.
발주Up 팀 드림
```

**제목**: K-apt 매일 들어가시는 분 계십니까? 저도 그랬습니다

```
방수공사 영업 10년 넘게 하다 보니 K-apt를 매일 새벽에 확인하는 게
습관이 됐습니다.

그런데 솔직히 공고가 뜨고 나서 연락하면 이미 늦더라고요.
일찍 관계 만들어 놓은 업체가 있으면 견적도 제대로 못 써봅니다.

그래서 알아봤더니 준공 연도, 누수 민원 기록, 장기수선 계획서만 봐도
대략 언제 발주할지 가늠이 된다는 걸 알게 됐습니다.

요즘은 그 데이터를 자동으로 정리해서 아침마다 보내주는
서비스(발주Up)를 베타로 써보고 있는데, 생각보다 쓸만합니다.

관심 있으신 분은 댓글 달아주시면 링크 공유해 드리겠습니다.
```

```

공고 뜨고 나서 전화하는 것과
공고 뜨기 전에 먼저 찾아가는 것,
수주율이 다릅니다.

이번 달 무료 베타 20명 모집합니다.
```

## 4. 인스타그램 광고

> 타겟 설정: 40~60대 / 건설업·방수·도장 관심사 / 소상공인·자영업 / 경기·서울

### 인스타 광고 A — 단일 이미지 (피드)

**비주얼 방향**: 아파트 단지 야경 사진 위에 텍스트 오버레이
(이미지 생성 시 지시: 서울 아파트 단지, 저녁 조명, 고요한 분위기, 텍스트 여백)

**헤드라인**: 공고 뜨기 전에 먼저 움직이십시오

**바디 카피**:
```
방수공사 수주는 공고보다 3~6개월 앞서 시작됩니다.

발주Up은 발주 임박 단지를 매일 아침 알림으로 보내드립니다.
수도권 무료 베타 모집 중.
```

**링크 문구**: 무료 체험 신청

---

### 인스타 광고 B — 카드뉴스형 (슬라이드 3장)

**1장**:
> 방수공사 영업, 공고 뜨고 나서 전화하십니까

**2장**:
> 공고가 뜨는 순간  
> 경쟁은 이미 시작된 후입니다  
>  
> 준공 후 10~15년이 지난 단지  
> 누수 민원이 증가한 단지  
> 장기수선 검토가 시작된 단지  
>  
> 이 신호를 먼저 읽는 쪽이 계약을 가져갑니다

**3장**:
> 발주Up  
> 발주 임박 단지를 매일 아침 알려드립니다  
>  
> 수도권 베타 무료 모집 중  
> 링크는 프로필에 있습니다

---

### 인스타 광고 C — 릴스/스토리 (15초 스크립트)

```
[0~3초] 자막: "공고 뜨고 전화하면 이미 늦습니다"
[3~8초] 자막: "발주Up은 공고 나오기 3~6개월 전, 발주 임박 단지를 매일 알림으로 보내드립니다"
[8~12초] 자막: "수도권 방수공사 업체 무료 베타 모집 중"
[12~15초] 자막: "프로필 링크에서 신청하십시오"

비주얼: 카카오 알림 도착 → 단지명 확인 → 전화 연결 화면 시퀀스
```
