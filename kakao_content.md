# 발주Up — 카카오톡 채널 콘텐츠 초안

> 톤 가이드: 전문적이되 딱딱하지 않게. 방수업체 대표 40~55세 눈높이.
> 이모지는 정보 구분용으로만 절제해서 사용 (메시지당 0~2개).
> 한 문장은 가급적 짧게, 모바일에서 한눈에 들어오게.

---

## 1. 채널 소개문 (100자 이내)

```
수도권 방수 발주, 공고 6개월 전에 알려드립니다.
1.7만 단지의 준공·이력·충당금을 분석한 발주 예측 리포트와
입찰 시작 알림을 받아보세요. (방수 전문업체 대상)
```

(공백 포함 99자)

> **변형 A — 더 짧게 (60자):**
> 수도권 1.7만 단지 발주 6개월 전 예측.
> 방수업체를 위한 영업 우선순위 정보를 매주 보내드립니다.

> **변형 B — 베네핏 강조:**
> 입찰 공고 뜨면 이미 늦습니다.
> 발주Up는 6개월 전에 알려드려, 경쟁 없이 단독 접촉할 시간을 만들어 드립니다.

---

## 2. 첫 공지 메시지 — 구독자 환영 + 얼리버드 안내

> 구독 직후 자동 발송. 1회만 노출되니 임팩트를 응축.

```
[발주Up] 구독 감사합니다

발주Up는 수도권 의무관리 아파트 1.7만 단지의
공공 데이터를 매주 분석해 "곧 방수 발주가 날 단지"를
6개월 전에 알려드리는 서비스입니다.

▷ 제공 정보
  - 단지별 발주 예측 점수 (0~100)
  - 장기수선충당금 잔액
  - 입찰공고 발생 실시간 알림
  - AI 자동 제안서 (관리소장 맞춤)

▷ 얼리버드 (선착순 100팀, 33자리 남음)
  - 월 12만 5천원 (정가 25만원, 50% 평생 할인)
  - 결제는 출시 후, 지금은 자리 예약만
  - 슈퍼 얼리버드 10팀은 마감

신청 → [얼리버드 신청하기]  ※ 랜딩 URL
문의 → 본 채널에 메시지 남겨주시면 답변드립니다.
```

## 3. 주간 발주 예측 단지 리스트 — 수동 발송 템플릿

> 매주 월요일 오전 9시 발송 권장. `[ ]` 자리를 그 주 데이터로 채워서 사용.
> 카카오 채널 길이 제한(1,000자)에 맞춰 단지 3~5개로 제한.

```
[발주Up] [YYYY년 N월 둘째 주] 주간 리포트

이번 주 발주 예측 점수가 가장 높은 단지 5곳을 공유드립니다.
지금 접촉하시면 입찰 공고 전 단독 협상이 가능합니다.

━━━━━━━━━━━━━━━━━━━━

1. [단지명]  ([시군구])
   점수 [92] · 준공 [2001]년 · [1,200]세대
   충당금 [3.2]억 · 마지막 방수 [2009]년

2. [단지명]  ([시군구])
   점수 [87] · 준공 [2003]년 · [780]세대
   충당금 [2.1]억 · 마지막 방수 [2010]년

3. [단지명]  ([시군구])
   점수 [84] · 준공 [2002]년 · [1,540]세대
   충당금 [4.1]억 · 마지막 방수 [없음]

4. [단지명]  ([시군구])
   점수 [82] · 준공 [2005]년 · [650]세대
   충당금 [1.8]억 · 마지막 방수 [2011]년

5. [단지명]  ([시군구])
   점수 [81] · 준공 [2004]년 · [920]세대
   충당금 [1.5]억 · 마지막 방수 [2010]년

━━━━━━━━━━━━━━━━━━━━

▷ 점수 의미
  80~100  즉시 접촉 (이번 분기 발주 가능성 높음)
  60~79   6개월 내 접촉 권장
  40~59   1년 내 모니터링

▷ 전체 단지 목록과 충당금 추이는 발주Up 앱에서
   확인 가능합니다. 정식 출시 시 안내드릴게요.
```

> **운영 팁:**
> - 단지명은 풀로 노출하지 않고 "OO아파트" 식 마스킹도 가능. 정식 가입자만 풀 데이터를 볼 수 있게 하면 가입 동기로 작용.
> - 발송 직후 5분간 채널 유입이 몰리니, 채팅 알림을 켜두세요.

## 4. 얼리버드 마감 D-3 알림 메시지

> 마감 3일 전 일괄 발송. 신청 안한 구독자만 타게팅 가능하면 더 좋음.

```
[발주Up] 얼리버드 마감 D-3

월 12만 5천원 평생 할인이 3일 뒤 종료됩니다.
이후로는 정가 월 25만원으로 적용됩니다.

남은 자리: [N]석 (총 100석 중)

▷ 지금 신청해야 하는 이유
  - 출시 직후 6개월간 정가의 50% 가격
  - 결제는 출시 후 진행 — 지금은 자리 예약만
  - 신청 후 언제든 취소 가능 (위약금 없음)
  - 슈퍼 얼리버드 10팀은 이미 마감

▷ 정가 적용 시 차액
  매월 12만 5천원 절약 = 연 150만원

신청 → [얼리버드 신청하기]  ※ 랜딩 URL
```

## 5. 출시 당일 공지 메시지

> 정식 서비스 오픈일 아침 발송.

```
[발주Up] 정식 출시 — 오늘부터 시작합니다

오랜 기다림 감사합니다.
발주Up가 오늘 정식으로 문을 엽니다.

▷ 얼리버드 가입자께
  - 등록하신 카카오톡 번호로 가입 안내 발송
  - 첫 로그인 후 결제 등록 (월 12만 5천원, 평생 동일가)
  - 발급된 ID로 수도권 1.7만 단지 전체 열람 가능

▷ 신규 가입자께
  - 7일 무료 체험 제공
  - 정가 월 25만원
  - 등록은 발주Up 웹에서

▷ 첫 주 사용 가이드
  1. 본인 주력 지역 필터 설정 (시·군·구)
  2. 점수 80점 이상 단지 5곳 선정
  3. 단지 상세 → AI 제안서 생성 (30초)
  4. PDF 다운로드 → 관리소장 접촉

문의는 본 채널 또는 앱 내 1:1 채팅으로 받습니다.
오늘 하루 좋은 수주 이어지시길 바랍니다.

→ [발주Up 바로가기]
```

## 메시지 작성 시 공통 가이드

- **숫자는 구체적으로**: "많은 단지" → "1.7만 단지", "할인" → "월 12만 5천원"
- **이모지는 정보 분리용으로만**: ▷ ━ 등 텍스트 구분자가 우선
- **링크는 메시지 끝에 1개만**: 클릭률이 가장 높음
- **존댓말 유지**, 단 과한 격식은 피함 ("드립니다" 정도가 적당)
- **이모티콘·일러스트 자제**: 40~55세 사업주 타겟에서는 신뢰도가 떨어짐
- **알림톡과 채널 메시지를 구분**: 알림톡은 1:1 트랜잭션(가입 확인 등), 채널 메시지는 1:N 마케팅
