# 사업 타당성 분석 보고서

**서비스명:** 발주Up (BaljuUp)
**분석 기준일:** 2026년 5월
**문서 유형:** 사업 타당성 분석 (Feasibility Study)

## Executive Summary

발주Up은 수도권 공동주택 방수공사 발주 시점을 AI로 예측해 **공고 6개월 전**부터 영업 타이밍을 알려주는 B2B SaaS 플랫폼입니다. 기존 K-apt 같은 사후 정보 제공 서비스와 달리, 시장 **선점(시간적 우위)**을 핵심 가치로 제공합니다.

### 핵심 수치 요약

| 항목 | 내용 |
|---|---|
| 타깃 시장 (TAM) | 수도권 공동주택 방수시장 약 **2.8조원** |
| 서비스 대상 단지 | 수도권 약 **1.7만 개** 아파트 단지 |
| 기준 요금제 | 프로 플랜 **월 11만원** (연 132만원) |
| 기본 시나리오 BEP | **약 303개사** (연 고정비 4억 기준) |
| 예측 근거 | 준공 **12년 차** 방수층 열화 통계 |
| 글로벌 시장 성장 | 2024년 340억 달러 → 2034년 633억 달러 (CAGR **6.7%**) |
| 국내 리모델링 시장 | 2025년 37조원 → 2030년 **44조원** |

### 종합 판단

발주Up은 **시장성·경쟁력·기술 타당성** 측면에서 모두 긍정적 평가를 받습니다. 다만 초기 고객 검증, AI 모델 정확도 검증, 법률 검토, 세부 비용 산출 등의 후속 작업을 통해 실행계획을 구체화해야 합니다.

**권장 추진 순서:** MVP 개발 → 파일럿 테스트 (10~20개사) → 고객 피드백 반영 및 모델 고도화 → 정식 출시

## 1. 서비스 기능 및 가치 제안

### 1.1 핵심 기능

발주Up의 핵심 기능은 영업의 각 단계를 자동화해 시공사의 수주 효율을 극대화하는 데 집중합니다.

**발주 예측 (핵심 차별화)**
수도권 약 1.7만 개 아파트 단지의 유지관리 이력·장기수선충당금·준공연도·세대수 등 다변수 데이터를 AI로 분석해 다음 방수공사 시점을 예측합니다. "공고 6개월 전" 정보를 제공해 경쟁 입찰 이전에 관리소장과 선접촉할 수 있게 합니다.

**우선순위 리스트**
예측 점수(0~100점)를 기반으로 단지를 순위화합니다. 시공사는 전문 분야(옥상·지하·외벽 등)에 맞춰 오늘 전화할 단지를 즉시 파악할 수 있습니다.

**실시간 알림**
관심 지역·단지 등록 시 신규 입찰공고·수의계약 공지를 카카오 알림톡으로 즉시 통보합니다. 대시보드에서 단지별 점수·연락처·공사 형태 등을 한 화면에 제공합니다.

**AI 제안서 자동 생성**
단지 정보(준공연도·세대수·충당금·과거 공사 이력)를 바탕으로 버튼 클릭 한 번에 PDF 제안서를 생성합니다. 수동 제안서 작성 대비 준비 시간을 90% 이상 단축합니다.

**데이터 자동화**
공공데이터포털(K-apt), 조달청, 국토교통부 API를 매일 자동 수집·분석해 최신 상태를 유지합니다. 신청된 API: 단지 목록·유지관리 이력·장기수선충당금·**입찰공고**·**수의계약 공지** (총 5종).

### 1.2 영업 워크플로우 혁신

```
[기존] 직접 검색 → 수동 정보 취합 → 제안서 작성(수 시간) → 전화 영업 → 결과 메모
[발주Up] 앱 접속 → 오늘의 추천 단지 확인 → 제안서 1분 생성 → 전화 → CRM 자동 기록
```

사용자는 **관심 지역 설정 → AI 자동분석 → 알림 → 제안서 작성 → CRM** 까지의 영업 워크플로우 전 단계를 하나의 플랫폼에서 처리할 수 있습니다.

### 1.3 우선순위별 개선 기능 로드맵

추가·강화하면 경쟁력이 높아지는 기능을 우선순위와 개발 난이도로 분류했습니다.

| **오픈 API 제공** | 건설사 ERP·CRM 연동용 외부 API | 중 | P2 |

| **결제/금융 연계** | 미수금 자동관리·소액결제 지원 | 높 | P3 |

## 2. 시장성 분석

### 2.1 시장 규모 (TAM / SAM / SOM)

| 지표 | 계산 기준 | 규모 |
|---|---|---|
| **TAM** (총시장) | 전국 공동주택 방수·유지보수 시장 × 수도권 비중(40%) | **2.8조원** |
| **SAM** (공략 가능 시장) | TAM × 타깃 단지 비중(상위 30%, 준공 10년 이상) | **0.84조원** |
| **SOM** (실현 가능 시장) | SAM × 목표 점유율(5%, 3년 내) | **420억원** |

> **전국 공동주택 방수·유지보수 입찰 시장** 은 약 7조원(2021년 기준)으로 추정됩니다. 수도권 아파트(약 1.7만 단지)가 전국의 약 40%를 차지합니다. SAM·SOM 비율은 초기 타깃 고객 수용 능력과 경쟁 상황을 보수적으로 반영한 예시이며, 실제 사업 전개에 따라 조정이 필요합니다.

SOM 420억원을 연 구독료(프로 플랜 기준 연 132만원)로 환산하면 약 **31,800개사** 규모입니다. 3년 내 목표는 이 중 300~500개사 확보로, 충분히 현실적인 범위입니다.

### 2.2 시장 성장 트렌드

**글로벌 방수 시장**
글로벌 방수 시장은 2024년 약 340억 달러에서 2034년 633억 달러로 성장(CAGR 6.7%)할 전망입니다. 빠른 도시화, 인프라 개발, 내구성 자재 수요 증가가 주요 성장 동인입니다. 특히 아시아태평양 지역이 두드러져 2035년 글로벌 점유율의 약 36%를 차지할 것으로 전망됩니다. 한국도 정부 주도 시설 개선 사업 및 신재생 에너지 인프라 확장으로 방수 수요가 지속 증가하고 있습니다.

**국내 리모델링·유지보수 시장**
1990년대 이후 공급된 단지들이 이제 본격적인 노후화 구간에 진입하면서 유지보수 수요가 급증하고 있습니다. 국내 건축물 리모델링(유지·보수 포함) 시장은 2020년 약 30조원에서 2025년 37조원, 2030년 44조원까지 확대될 전망입니다. 방수공사는 전체 유지보수 공사 중 옥상(약 60%), 지하(약 30%)를 차지하는 안전 핵심 분야입니다.

발주Up이 예측 근거로 삼는 **"준공 12년 차 방수층 열화"** 통계는 건설 업계 실증 데이터와 일치합니다. 1990년대에 지어진 단지들이 현재 30년 이상이 되어 재방수 수요도 동시에 발생하고 있어, 시장 타이밍이 매우 유리합니다.

### 2.3 규제·정책 환경

**우호적 요인**
공동주택관리법 개정으로 방수공사 등 대형 유지보수 공사의 전자입찰 의무화가 확대되고 있습니다. 이는 공공 데이터 가용성을 높이고, 발주Up의 예측 모델 데이터 품질을 지속 개선하는 긍정적 요인입니다. 공공데이터포털 API는 상업적 활용이 명시적으로 허용되어 라이선스 리스크가 낮습니다.

정부의 AI·창업 지원 제도(초기창업패키지, R&D 지원, 스케일업 프로그램 등)도 적극 활용 가능합니다. 2026년부터 창업 지원에 ESG 평가가 도입되어, AI 기반 친환경 방수 솔루션을 내세우면 지원사업 선정 시 가점을 기대할 수 있습니다.

**준수 필요 사항**
민간 데이터(관리소장 연락처 등)는 개인정보보호법에 따른 수집·이용 동의 절차를 반드시 거쳐야 합니다. 관련 법률 자문을 초기에 받아두는 것이 권장됩니다.

## 3. 수익성 분석

### 3.1 수익 모델

발주Up은 B2B SaaS 월 구독 기반입니다. 핵심은 **낮은 구독료 대비 압도적인 고객 ROI**입니다. 방수공사 1건 수주 금액(평균 3,000만~1억원)이 연 구독료(132만원)의 수십 배이므로, 단 1건의 추가 수주만으로도 구독료 이상의 투자회수가 가능합니다.

**요금제 구조 (안)**

| 플랜 | 월 요금 | 연 요금 | 주요 기능 |
|---|---|---|---|
| **스타터** | 55,000원 | 660,000원 | 단지 조회·알림, 제안서 5건/월 |
| **프로** | 110,000원 | 1,320,000원 | 무제한 조회·알림, 제안서 무제한, CRM |
| **엔터프라이즈** | 협의 | 협의 | 팀 계정, API 연동, 전담 CS |

**부가 수익원**
구독 외에도 빅데이터 분석 리포트 판매, VIP 컨설팅, ERP/회계 시스템 연동 플러그인 등으로 ARPU(가입사당 평균 매출)를 높일 수 있습니다.

### 3.2 비용 구조

**고정비 (연간 추정)**

| 항목 | 연간 비용 (추정) |
|---|---|
| 개발·운영 인력 (3~5인) | 2.0억~3.5억원 |
| 클라우드 인프라 (Vercel, Supabase 등) | 0.1억~0.3억원 |
| 마케팅·광고 | 0.3억~0.5억원 |
| 기타 (법무, 사무, 도구) | 0.1억~0.2억원 |
| **합계** | **2.5억~4.5억원** |

**변동비**
알림톡 발송비(건당 약 15~20원), 클라우드 컴퓨팅(데이터 수집·AI 추론), 고객 지원 비용 등이 가입사 수에 비례해 증가합니다.

### 3.3 손익분기점 (BEP) 시나리오

> **계산 방식:** BEP(개사) = 연간 고정비 ÷ 가입사당 연 구독료

| 시나리오 | 기준 요금 | 연 구독료 | 연 고정비 | BEP 가입사 수 |
|:---:|:---:|:---:|:---:|:---:|
| **보수적** | 스타터 월 5.5만 | 66만원 | 4.5억원 | **≈ 682개사** |
| **기본** | 프로 월 11만 | 132만원 | 4억원 | **≈ 303개사** |
| **공격적** | 프로+부가 월 22만 | 264만원 | 3억원 | **≈ 114개사** |

기본 시나리오(BEP 303개사) 기준으로, 월 10개사씩 신규 유치하면 약 **30개월(2.5년)** 내 BEP 도달이 가능합니다. 공격적 시나리오에서는 영업 파트너십이나 컨설팅 패키지 판매 시 ARPU가 높아져 BEP가 크게 낮아집니다.

### 3.4 수익 성장 시뮬레이션 (프로 플랜 기준)

```
      가입사 수    연 매출      연 고정비    영업이익
1년차   50개사    0.66억원      4.0억원    -3.34억원
2년차  150개사    1.98억원      4.0억원    -2.02억원
3년차  303개사    4.00억원      4.0억원      BEP 도달
4년차  500개사    6.60억원      4.5억원    +2.10억원
5년차  800개사   10.56억원      5.0억원    +5.56억원
```

### 3.5 민감도 분석

구독료·판매 속도·비용 변화가 BEP에 미치는 영향:

- **구독료 10% 인상** → BEP 가입사 수 약 9% 감소 (이탈 위험 낮음, 1건 수주로 충분히 커버)
- **구독료 10% 인하** → BEP 가입사 수 약 11% 증가 (도입 장벽 낮추는 효과)
- **고정비 10% 증가** → BEP 가입사 수 약 10% 증가
- **월 신규 유치 +5개사** → BEP 도달 약 6개월 단축

가장 임팩트가 큰 레버는 **ARPU 향상**(부가 수익 판매)과 **초기 이탈률 관리**(온보딩 품질)입니다.

## 4. 경쟁 분석

### 4.1 경쟁 구도

| 경쟁사 | 제공 서비스 | 강점 | 약점 |
|---|---|---|---|
| **K-apt** | 공동주택 입찰공고·결과·장기계획 조회 | 공신력, 무료, 법적 의무 시스템 | 사후 정보, 영업 자동화 부재 |
| **KBID** | 전국 입찰공고 검색·알림 | 방대한 DB(공공·민간 모두), 인지도 | AI 예측 없음, 비용 높음 |
| **i건설넷** | 입찰정보 알림, 교육 세미나 | 맞춤 알림, 초기 사용자 기반 | 예측 기능 부족, 업데이트 한계 |
| **나라장터** | 공공 전자입찰 시스템 | 공공 발주 직접 연계 | 민간 관리공사 정보 부족 |
| **발주Up** | AI 발주 예측 + 알림 + 제안서 + CRM | **6개월 선점 예측**, 영업 All-in-one | 초기 브랜드 인지도 부족 |

### 4.2 핵심 차별화 포인트

발주Up과 기존 경쟁사의 결정적 차이는 **'언제 영업하느냐'** 에 있습니다.

```
[기존 서비스]  공고 게시 → 모든 업체가 동시에 확인 → 가격 경쟁 → 저마진 수주
[발주Up]      AI 예측 → 공고 6개월 전 선접촉 → 신뢰 관계 구축 → 유리한 조건으로 수주
```

K-apt와 KBID는 **"공고를 보여주는" 도구**입니다. 발주Up은 **"공고가 나기 전에 움직이게 하는" 도구**입니다. 이 시간적 우위가 발주Up의 핵심 방어벽(Moat)입니다.

### 4.3 차별화 전략

**포지셔닝 메시지**
> *"K-apt는 확인용, 발주Up은 선점용"*
> *"공고 뜨고 움직이면 늦습니다"*

**성과 중심 마케팅**
단순 조회수가 아닌 수주 성공률 향상을 KPI로 제시합니다. 초기 파일럿 고객의 수주 성공 사례를 수치로 공개해 레퍼런스 마케팅에 활용합니다.

**초기 유인 전략**
3개월 무료 체험 후 전환을 유도하고, 연간 결제 시 2개월 무료 혜택을 제공합니다. 초기 50개사는 직접 영업으로 확보하고, 이후 건설업 커뮤니티·협회를 통한 채널 판매로 전환합니다.

## 6. 결론 및 권고

### 6.1 종합 평가

시장·수익성·경쟁력 분석 결과, **발주Up은 높은 사업 타당성을 갖춘다**고 판단합니다.

**시장 잠재력 ✓**
글로벌 방수 시장 CAGR 6.7%, 국내 리모델링 시장 2030년 44조원 확대, 수도권 노후 아파트 방수공사 수요 급증이라는 세 가지 성장 동력이 맞물려 있습니다.

**수익 가능성 ✓**
월 11만원의 구독료는 고객에게 극히 낮은 리스크로, 단 1건의 추가 수주만으로도 ROI가 수십 배에 달합니다. LTV(고객 생애 가치) 또한 방수 업계의 반복 발주 특성상 높게 유지될 가능성이 큽니다.

**경쟁 우위 ✓**
"공고 6개월 전 선점 예측"은 국내 어떤 경쟁 서비스도 제공하지 않는 독보적 기능입니다. 해외 대형 SaaS는 프로젝트 관리 중심이어서 국내 공동주택 방수공사 특화 시장에 단기 진입 가능성이 낮습니다.

### 6.2 단기 실행 로드맵

**Phase 1 (0~3개월): MVP 검증**
- 핵심 기능 3가지에 집중: 예측 리스트 + 알림 + 제안서 생성
- 파일럿 고객 10~20개사 무료 유치 (수도권 방수 시공사)
- 주 1회 사용 인터뷰로 UX 개선 반영

**Phase 2 (3~9개월): 초기 성장**
- 유료 전환 및 정식 요금제 적용
- CRM·이메일 자동 전송 기능 추가 (P1 항목)
- 건설업 협회·커뮤니티 채널 파트너십 구축
- 정부 지원금(초기창업패키지, TIPS) 신청

**Phase 3 (9~18개월): 확장**
- 수도권 전체 → 광역시 확장
- 방수 외 도장·조경 등 인접 공종으로 서비스 확대
- 엔터프라이즈 플랜 출시 (팀 계정, API)

### 6.3 후속 조사 필요 사항

사업 계획 구체화를 위해 다음 데이터를 추가 확보해야 합니다.

- **고객 인터뷰**: 타깃 시공사 10개사 이상, 현재 영업 방식·예산·도구 사용 현황 파악
- **AI 예측 모델 검증**: 과거 3년치 실제 발주 데이터로 예측 정확도 백테스트
- **법률 자문**: 개인정보보호법·공동주택관리법 준수 체크리스트 확보
- **경쟁사 요금 조사**: KBID·i건설넷 실제 요금제 및 해지율 파악
- **세부 비용 산출**: 인력 채용 계획 확정 후 정밀 고정비 산출

## 7. 개발 제약 및 디자인 원칙

### 7.1 이미지 사용 금지 원칙

> **[앱 전반 규칙] 이미지(img 태그, 외부 이미지 URL, base64 임베드 포함) 사용 절대 금지**

발주Up 앱 전반에서 이미지를 사용하지 않는다. 이 규칙은 모든 페이지, 컴포넌트, 이메일 템플릿, PDF 출력물, 랜딩페이지에 동일하게 적용된다.

**적용 범위:**
- Next.js 앱 전체 (`/app`, `/components`, `/pages`)
- 랜딩페이지 HTML
- PDF 제안서 출력 (`@react-pdf/renderer`)
- 이메일 템플릿 (Resend)
- 카카오 알림톡 메시지
- 운영 대시보드

**대체 수단:**
- 아이콘: SVG 인라인 또는 텍스트 기호 (✓ ✗ → ← 등)
- 로고/브랜드: 텍스트 + CSS 스타일링
- 차트/그래프: CSS 기반 또는 순수 텍스트 테이블
- 일러스트: CSS 애니메이션 또는 Unicode/Emoji

**이유:** 초기 MVP 단계에서 이미지 에셋 관리 비용 제거, 로딩 속도 최적화, Vercel 배포 용량 최소화.
