# 발주Up — 사전 모집 & 마케팅 전략
> 타겟: 40~60대 건설업 방수공사 영업담당자 / 중소업체 대표  
> 단계: 베타 사전 모집 → 정식 출시

---

## 1. 타겟 이해 — 40~60대 영업담당자

### 행동 패턴
| 항목 | 특성 |
|---|---|
| 주 정보 채널 | 카카오톡, 네이버, 유튜브, 지인 소개 |
| 의사결정 방식 | 직접 써봐야 믿음 / 복잡한 가입 기피 |
| 신뢰 기반 | 아는 사람 추천 > 광고 > 인터넷 후기 |
| 디지털 수준 | 스마트폰 능숙 / PC 덜 선호 |
| 구매 저항 포인트 | "이게 진짜 되나?", "비싸면 안 쓴다" |

### 핵심 심리
- **손해 공포**: "경쟁사가 이미 쓰고 있으면?" → 조바심 유발
- **체면 문화**: "앞서가는 영업인"이라는 자부심 자극
- **관계 중시**: 단순 SaaS가 아닌 "영업 파트너" 포지셔닝
- **단순함 선호**: 기능 설명보다 "아침에 받고 전화만 하면 됩니다" 한 줄이 효과적

---

## 2. 사전 모집 전략 (베타, 0~3개월)

### 목표
- 베타 사용자 **30~50명** 확보
- 실사용 피드백 수집 → 정식 출시 전 제품 완성도 제고

---

### Phase 1 — 직접 아웃리치 (1~4주)

**가장 확실한 첫 번째 채널: 직접 연락**

40~60대는 광고보다 직접 전화·문자에 반응률이 높습니다.

#### 실행 방법

1. **타겟 리스트 구축**
   - 방수공사업 등록 업체 조회 (건설업 면허 정보 공개 데이터)
   - 지역: 서울·경기 우선 (서비스 커버리지 일치)
   - 규모: 직원 5~30인 중소업체 우선

2. **첫 접촉 — 카카오톡 문자**
   ```
   안녕하세요, 발주Up 팀입니다.
   
   방수공사 영업하실 때 K-apt 매일 확인하십니까?
   저희는 공고 나오기 3~6개월 전에 발주 임박 단지를
   카카오 알림톡으로 보내드리는 서비스입니다.
   
   이번 달 수도권 베타 사용자 20명을 무료로 모집합니다.
   관심 있으시면 아래 링크 한 번만 눌러봐 주십시오.
   [링크]
   ```

3. **전화 팔로업** (문자 발송 2일 후)
   - "문자 받으셨나요?" 한 마디로 시작
   - 3분 안에 핵심 전달: "아침에 알림 받고, 전화하고, 관계 만드는 것"
   - 무료 체험 강조, 부담 없음 명시

---

### Phase 2 — 업계 커뮤니티 침투 (2~6주)

#### 네이버 카페 공략
- **타겟 카페**: 방수공사, 도장공사, 건설업 영업, 아파트 리모델링 관련 카페
- **접근법**: 광고 게시 X → 정보성 글로 신뢰 쌓기 후 서비스 소개
- **게시글 전략**:
  ```
  제목: "K-apt 매일 들어가는 분 계십니까? 저도 그랬습니다"
  
  내용: 공감 스토리 → 문제 제기 → 발주Up 자연스럽게 소개
  → 댓글로 베타 신청 안내
  ```

#### 밴드(BAND) 활용
- 건설업 종사자, 중소기업 대표 밴드 다수 활성화
- 업계 정보 공유 형식으로 진입
- "수도권 방수공사 발주 임박 단지 정보 공유합니다" 형식의 정기 포스팅

#### 카카오오픈채팅
- "방수공사 영업 정보 나누는 방" 직접 개설
- 무료 단지 정보 정기 제공 → 자연스럽게 발주Up 서비스로 연결

---

### Phase 3 — 협회 & 파트너십 (4~8주)

| 기관 | 접근 방법 |
|---|---|
| 한국방수공사협회 | 회원사 대상 협력 세미나 또는 소식지 게재 |
| 대한건설협회 지부 | 지역 모임 참석, 명함 교환, 서비스 소개 |
| 방수재료 제조사 | 영업팀과 파트너십 → 고객 공동 개발 |
| 건설공제조합 | 회원 혜택으로 발주Up 베타 연계 제안 |

## 3. 채널별 마케팅 전략 (정식 출시 준비)

### 채널 우선순위 (40~60대 기준)

```
1순위: 카카오톡 (접근성 최고)
2순위: 네이버 (검색 의존도 높음)
3순위: 유튜브 (사용 증가 추세)
4순위: 지인 소개 (신뢰도 최고)
5순위: 인스타그램 (보조 채널)
```

---

### 카카오 채널 운영

**카카오 비즈니스 채널 개설 필수**

- 채널명: `발주Up 공식`
- 친구 추가 유도 → 카카오 알림톡 발송 채널과 연동
- **주 1회 콘텐츠**: "이번 주 수도권 발주 주목 단지" 리포트 (무료 공개)
- **알림톡 템플릿**: 베타 신청자에게 "발주Up 알림 서비스 시작 안내"

#### 카카오 광고 (카카오모먼트)
- 타겟: 40~60대 남성, 건설업 관련 키워드 관심사
- 소재: "공고 나오기 전에 연락하는 영업팀이 계약을 가져갑니다"
- 랜딩: 발주Up 베타 신청 페이지

---

### 네이버 전략

#### 네이버 블로그 SEO
검색 키워드 타겟:
- "방수공사 영업 방법"
- "K-apt 활용법"
- "아파트 방수공사 발주 시기"
- "공동주택 방수공사 수주"

콘텐츠 방향:
- 정보성 글 위주 (광고처럼 보이면 이탈)
- "아파트 방수층 수명은 몇 년인가" 같은 실무 정보 제공
- 마지막에 자연스럽게 발주Up 언급

#### 네이버 검색광고
- 키워드: "방수공사 발주", "공동주택 방수", "K-apt 공고"
- 예산: 월 20~30만 원 (초기 테스트)

---

### 유튜브 전략

40~60대의 유튜브 사용률은 빠르게 증가 중입니다.

**타겟 채널**: 건설업, 소상공인, 영업 노하우 관련 유튜브 채널
- 인플루언서 협업: 건설업 유튜버에게 서비스 소개 의뢰 (체험 형식)
- 광고 노출: 건설/부동산/영업 관련 영상에 인-스트림 광고

**자체 콘텐츠 (장기)**:
- "방수공사 수주율 높이는 영업 루틴" 시리즈
- 실제 베타 사용자 인터뷰 영상

---

### 소개 프로그램 (Referral)

40~60대는 지인 추천을 가장 신뢰합니다.

**설계안**:
- 기존 사용자가 동료 소개 → 양쪽 모두 1개월 무료 연장
- 소개 링크는 카카오톡으로 바로 공유 가능하게 구성
- "우리 팀장님이 쓰는 서비스" 형식의 바이럴 유도

## 4. 메시지 전략 — 연령대별 언어

### 피해야 할 표현
- "AI 기반", "머신러닝", "데이터 파이프라인" → 거부감
- 과도한 영어 조어, 스타트업 용어
- 복잡한 기능 나열

### 효과적인 표현
| 피해야 할 표현 | 대신 이 표현 |
|---|---|
| "AI 예측 알고리즘" | "공고 나오기 전에 먼저 알려드립니다" |
| "데이터 기반 인사이트" | "아파트 준공 연도 보면 다 나옵니다" |
| "온보딩 프로세스" | "가입하고 내일 아침부터 바로 받아보십시오" |
| "SaaS 구독 모델" | "월정액으로 쓰고, 언제든 해지 가능합니다" |

### 핵심 메시지 (한 문장)
> **"공고 뜨기 전에 전화한 업체가 계약을 가져갑니다."**

## 5. 실행 타임라인

```
Week 1~2   : 타겟 리스트 100개 구축 → 직접 문자/전화 아웃리치
Week 3~4   : 네이버 카페, 밴드 커뮤니티 진입 / 카카오 채널 개설
Week 5~6   : 협회 접촉, 파트너십 논의 시작
Week 7~8   : 베타 사용자 피드백 수집 → 제품 보완
Week 9~10  : 소개 프로그램 론칭 / 네이버 블로그 콘텐츠 축적
Week 11~12 : 정식 출시 준비 (카카오 광고, 네이버 광고 테스트)
```

---

## 6. 베타 모집 성공 지표

| 지표 | 목표 |
|---|---|
| 베타 신청자 | 50명 이상 |
| 실제 사용자 (로그인) | 30명 이상 |
| 주간 재방문율 | 60% 이상 |
| 카카오 알림톡 수신 후 연락 시도율 | 40% 이상 |
| 만족도 (5점 만점) | 4.0 이상 |
| 정식 출시 시 유료 전환 의향 | 50% 이상 |

## 7. 주의사항

- **과도한 데이터 공개 금지**: 실제 단지명, 고객사, DB 구조 노출 X (카피캣 방어)
- **허위 과장 광고 금지**: "100% 정확도", "반드시 수주" 등의 표현 삼가
- **개인정보 처리**: 베타 신청자 정보는 개인정보처리방침 기준으로 관리
- **기대치 관리**: "예측 서비스"임을 명확히 → 발주 보장이 아님을 고지
